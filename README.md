# Todo List Dashboard

A modern todo list application with frontend and backend.

## Tech Stack
- **Backend**: Node.js + Express
- **Frontend**: React + TypeScript
- **Database**: SQLite (development) / PostgreSQL (production)

## Setup

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
